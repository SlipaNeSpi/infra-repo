# Инфраструктурный репозиторий (FluxCD)

Репозиторий содержит конфигурацию для управления кластером Kubernetes через механизм GitOps с использованием FluxCD.

## Предназначен для:
1) Хранение манифестов для установки и управления компонентами FluxCD
2) Определение источников синхронизации (инфраструктурного и прикладного репозиториев)
3) Управление желаемым состоянием кластера через версионирование в Git
